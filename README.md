# Полдень — самарский городской журнал

## Что изменилось

Сайт больше не агрегирует RSS сторонних новостных изданий (РИА, ТАСС,
Lenta.ru, Коммерсантъ, BBC Russian). Теперь это полностью авторский
проект: 20 собственных материалов о жизни Самары, которые можно
свободно публиковать и показывать целиком — прямо на сайте, без
"читать полностью на источнике".

## Структура

```
nextjs-news/
  lib/
    articles.js      ← все 20 статей: заголовок, категория, дата, текст
    i18n.js           ← словарь интерфейса RU/EN
    format.js         ← форматирование времени
  app/
    layout.js         ← шапка + курс валют (ЦБ РФ)
    context/LanguageContext.js  ← RU/EN, кэш переводов
    components/
      SiteChrome.js   ← шапка, меню разделов, бегущая строка
      NewsFeed.js     ← лента статей (главная и категории)
      ArticleBody.js  ← текст статьи с поддержкой перевода
      ShareButtons.js ← скопировать ссылку / Telegram
    page.js                    ← главная страница
    news/[category]/page.js    ← страница раздела
    article/[slug]/page.js     ← страница отдельной статьи (полный текст)
    api/translate/route.js     ← перевод на английский (MyMemory API)
  package.json
```

## Категории

Новости · Город · Культура · Еда · Путешествия · Спорт — статьи
распределены по темам вручную в `lib/articles.js`.

## Фото

Сейчас у всех статей `image: null` — вместо фото показывается тёплая
заглушка с первой буквой заголовка. Когда появятся фотографии — пришли
их, и я подставлю пути вида `/public/images/...` в поле `image`
каждой статьи.

## Как добавить статью

Просто добавь новый объект в массив `ARTICLES` в `lib/articles.js`:
slug, title, category (одна из шести выше), pubDate, image, excerpt,
body (массив абзацев). Страницы `/article/[slug]` и списки категорий
подхватят её автоматически — руками больше ничего создавать не нужно
благодаря `generateStaticParams`.
